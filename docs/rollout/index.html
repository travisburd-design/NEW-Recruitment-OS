What you need to set up

The toolkit — accounts to provision

All proven, low-cost, mostly tools you may already own. The Google Sheet + Apps Script is the application — everything else feeds it.

Core engine

Google Workspace

Sheets, Apps Script, Gmail, Forms, Calendar, Drive. The whole OS runs here. Likely already in place.

Intelligence

Google Gemini API

Scores pre-screens, grades interview transcripts, drafts reports. Pay-as-you-go — pennies per candidate.

Transcripts · in-person

Otter.ai

Records & transcribes in-person interviews; piped in automatically for AI grading.

Transcripts · online

Fathom

Captures online/video interviews. Generous free tier.

Automation glue

Zapier

Moves Otter transcripts into the sheet. One org-level account covers all locations.

Scheduling

Koalendar (or Calendly)

Candidate self-booking links for phone, live, and working interviews.

Candidate sources

Indeed / ACT Auto Staffing

Job leads flow in by email and are auto-imported & invited. Ad spend is pay-for-performance.

Optional

Custom domain

For a branded oversight dashboard / this very report. ~$12/year.

From one shop to thirty-three

The rollout playbook

The hard part — building the system — is done. Each new location is a repeatable clone, not a rebuild.

Package the master template

The current Recruiting OS becomes the gold-copy: Sheet + Apps Script + Forms + config. ✓ Already built.

Provision shared accounts

One-time: Gemini API key, Zapier, Otter/Fathom, Koalendar at the org/district level.

Clone per location

Copy the master Sheet, paste the Apps Script, link the 5 Google Forms. ~30–45 min each.

Configure the shop

Fill the Config tab: shop identity, active roles, hiring manager, booking links, pay ranges.

Bootstrap & smoke-test

Run Bootstrap, Seed Templates, Install Triggers; verify end-to-end in safe TEST mode.

Set up the manager view

One click: “Set Up Tabs for New Manager” — 10 clean tabs, color-coded, the rest hidden.

Train in 15 minutes

The “GM Daily” card teaches the whole job: read the email → pick a dropdown. Then GO LIVE.

Scale across the district

Repeat the clone. Monitor every shop from the twice-daily digests. Roll to all 33.
